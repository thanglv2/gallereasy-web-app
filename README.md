# Gallereasy
This is a web app to aggregate results from various online image libraries.

This app was built from scratch with:
- React v16
- React Router v5
- Styled-components
- Jest, Enzyme
- Webpack v4, Babel v7

# Compatibility
- Support Internet Explorer 11 and upwards
- Responsive layout and mobile-friendly
